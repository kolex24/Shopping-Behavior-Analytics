# Shopping-Behavior-Analytics
The Shopping behavor analytics project focuses on  how customers interact with retails product across differents locations. By analyzing transactional and demographic data,
the project identifies key customer segments, purchasing patterns, and product affinities. these insights help businesses make informed decisions about marketing and customer
retention strategies.

## Project Objectives
This project aims to:
- Analyze customer shopping behavior and purchase trends.
- Understand product category perfformance and shopping frequency.
- Build 3 pages visual dashboard for business decision making.
- Recommend data-driven strategies to improve customer engagement and purchase.
